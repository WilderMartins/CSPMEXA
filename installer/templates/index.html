Assistente de Instalação do CSPMEXA

Bem-vindo! Preencha os campos abaixo para configurar e iniciar seu ambiente CSPMEXA. Valores padrão já estão preenchidos para facilitar.

Passo 1: Configuração do Banco de Dados

Credenciais para o banco de dados PostgreSQL onde as informações de usuários e políticas serão armazenadas.

Passo 2: Configurações Gerais e de Segurança

Portas de serviço e chaves de segurança. A chave secreta JWT será gerada automaticamente para sua segurança.

Será gerada automaticamente no servidor.

Passo 3: Login com Google (Opcional)

Para permitir login com contas Google, insira as credenciais OAuth 2.0 do seu projeto no Google Cloud. Você pode pular esta etapa.
